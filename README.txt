o iCake é uma aplicação desenvolvida com cakePHP 2.1, jQuery 1.5 e mysql 5.1.
Recomendável para usuários iniciantes que estão estudando o cakePHP, mas também útil
a usuários avançados que precisem de algo além da agilidade que o cakePHP já nos oferece.

Se você teve acesso a este código e quer retribuir com ajuda, leia o arquivo Docs/pendências.txt

A instalação é fácil e intuitiva, basta descompactar este código no seu servidor web, algo
como /var/www e acesse http://localhost/icake

Se estiver usando windows com o xammp, o seu diretório deve estar em
c:\xampp\htdocs

Entre com login e senha qualquer, após o erro de login, será solicitado:
- a instalação do banco de dados
- a instalação do usuário administrador

Maiores detalhes da aplicação, leia os arquivos do diretório 'Docs'.

Espero que este código lhe seja útil.
